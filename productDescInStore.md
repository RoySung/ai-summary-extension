AskWeb AI is a fast and flexible browser extension that helps you understand any web page in seconds.

Use your own Gemini or OpenAI API key to generate full-page summaries, summarize selected text, and ask follow-up questions based on the page content. Open AskWeb AI from the extension icon or the floating button, then read and chat in a clean resizable window or a dedicated full-page view.

Why AskWeb AI:
• Bring your own API key — no subscription or hidden fees
• Summarize entire pages or selected text
• Ask questions with page-aware context
• Support for Gemini and OpenAI models
• Floating window and full-page reading modes
• Custom prompts and theme options

Whether you are reading articles, research, documentation, or long-form content, AskWeb AI helps you get the key points faster and stay focused.

Security & Privacy:
• All processing happens locally in your browser — your data is never stored or sent to third-party servers (except Gemini/OpenAI APIs you configure)
• Open source: Review the code at https://github.com/RoySung/ai-summary-extension
