Privacy Policy for AskWeb AI

Last Updated: March 16, 2026

AskWeb AI ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how our Chrome Extension collects, uses, and discloses information.

1. Information Collection and Use

  • Web Page Content: When you click "Summarize" or interact with the extension, we extract the text content of your currently active tab.
  • Third-Party APIs: This text content is sent directly to your configured third-party AI provider (e.g., Google Gemini or OpenAI) to generate summaries or answer questions. We do not store or transmit this data to our own servers.
  • User Settings and API Keys: Your API keys and extension preferences are stored locally on your device using your browser's local storage sync mechanism. We do not have access to your API keys.

2. Data Retention

We do not collect, log, or store your web browsing history, the content you summarize, or any personally identifiable information on any external servers. All processing is either done locally or sent directly from your browser to the third-party AI provider you choose.

3. Contact Us

If you have any questions about this Privacy Policy, please contact us by creating an issue in our GitHub Repository.

AskWeb AI 隱私權政策

最後更新日期：2026年3月16日

AskWeb AI（以下簡稱「我們」）致力於保護您的隱私。本隱私權政策說明我們的 Chrome 擴充功能如何收集、使用與處理資訊。

1. 資訊收集與使用

  • 網頁內容：當您點擊「摘要」或與擴充功能互動時，我們會擷取您當前活動分頁的文字內容。
  • 第三方 API 傳輸：這些文字內容會直接傳送至您所設定的第三方 AI 服務提供者（例如 Google Gemini 或 OpenAI）以產生摘要或回答問題。我們不會將這些資料儲存或傳送到我們的伺服器。
  • 使用者設定與 API 金鑰：您的 API 金鑰和擴充功能偏好設定皆使用瀏覽器的本地儲存同步機制，安全地儲存在您的裝置上。我們無法存取您的 API 金鑰。

2. 資料保留

我們不會在任何外部伺服器上收集、記錄或儲存您的網路瀏覽紀錄、您摘要的內容，或任何個人識別資訊。所有的處理過程皆在本地完成，或是直接從您的瀏覽器傳送至您選擇的第三方 AI 提供商。

3. 聯絡我們

如果您對本隱私權政策有任何疑問，請透過我們的 GitHub Repository 建立 Issue 來聯絡我們。
