# AskWeb AI

AskWeb AI 是一款強大且優雅的 Chrome 擴充功能，可利用 AI（Google Gemini 或 OpenAI）為任何網頁進行摘要與互動式問答。

## ✨ 主要功能

- **🆓 100% 免費 & 支援自帶金鑰 **：沒有訂閱費或任何隱藏費用。使用您自己的 Google Gemini 或 OpenAI API 密鑰，直接與服務商結算（或直接享受他們的免費額度）。
- **🚀 即時網頁摘要**：只需點擊一下，即可獲得任何網頁的簡明準確摘要。
- **💬 互動式問答對話**：可與 AI 進行關於網頁內容的對話。詢問特定問題並根據上下文獲得解答。
- **🎯 懸浮球模式**：透過不干擾閱讀的懸浮球快速存取摘要功能。點擊後可展開為**可調整大小的視窗**，讓您邊瀏覽邊閱讀摘要。
- **📄 全螢幕檢視**：切換到專屬的沉浸式檢視模式，適合閱讀長篇摘要或管理複雜的問答內容。
- **🎨 佈景主題切換**：透過內建主題自訂介面外觀

## 🤖 支援的 AI 服務商

- **Google Gemini**（推薦免費用戶使用）
- **OpenAI**

## 🛠️ 安裝方式

### 開發模式

1. 複製此存放區（Clone this repository）。
2. 安裝依賴套件：
    ```bash
    pnpm install
    ```
3. 啟動開發伺服器：
    ```bash
    pnpm dev
    ```
    Chrome 將會自動開啟並載入擴充功能。

### 生產版本建置

1. 建置擴充功能：
    ```bash
    pnpm build
    ```
2. 載入至 Chrome：
    - 開啟 `chrome://extensions/`
    - 開啟右上角的**開發人員模式**。
    - 點擊**載入未封裝項目**。
    - 選擇 `.output/chrome-mv3` 目錄。

## ⚙️ 設定與組態

1. **開啟擴充功能**：點擊擴充功能圖示或網頁上的懸浮球。
2. **前往設定**：點擊標題列中的齒輪圖示 (⚙️)。
3. **選擇服務商**：選擇 Google Gemini 或 OpenAI。
4. **輸入 API 金鑰**：
    - **Gemini**: 前往 [Google AI Studio](https://aistudio.google.com/apikey) 取得。
    - **OpenAI**: 前往 [OpenAI Platform](https://platform.openai.com/api-keys) 取得。
5. **選擇主題**：選擇您偏好的視覺風格（溫暖或冷色）。
6. **自訂提示詞**（選填）：自訂摘要與問答的系統提示詞，以符合您的特殊需求。
7. **懸浮球**：您可以依需求在特定網頁上切換懸浮球的顯示狀態。

## 📖 使用指南

### 使用懸浮視窗（總結整頁內容）

1. 點擊瀏覽器工具列上的**擴充功能圖示**或是直接點擊網頁上的**懸浮球**，即可打開視窗。
2. **調整大小**：拖曳視窗邊緣可依喜好調整大小。
3. **產生摘要**：點擊「✨ 總結網頁」開始總結整頁全篇內容。
4. **聊天問答**：在輸入框中輸入文字，提出後續相關問題。

### 總結特定選取範圍

在網頁上反白選取特定的文字段落，即可僅針對該選取範圍進行重點總結，不必總結整篇文章。

### 全螢幕檢視

若想深入了解，點選摘要檢視畫面中的「**📄 Full Page**」按鈕。這將會在一個專屬的獨立標籤頁中開啟您當前的摘要與聊天紀錄，提供更多空間讓您思考和閱讀。
